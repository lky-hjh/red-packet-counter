### 1. 上传部署包到服务器
使用FTP、宝塔面板文件管理器或其他方式，将 `red-packet-counter-deploy.zip` 上传到服务器Web根目录（如：`/www/wwwroot/`）

### 2. 解压部署包
在服务器上执行：
```bash
unzip red-packet-counter-deploy.zip -d red-packet-counter
```

### 3. 配置宝塔面板
1. 登录宝塔面板（http://175.178.84.235:8888）
2. 进入"网站"管理
3. 选择要替换的网站或创建新网站
4. 设置网站根目录为：`/www/wwwroot/red-packet-counter/dist`
5. 保存配置

### 4. 测试访问
- 打开浏览器访问：http://175.178.84.235
- 确认红包计数器正常显示

## ✨ 功能特性
✅ **六个红包金额按钮**：1元、5元、10元、20元、50元、100元  
✅ **实时金额汇总**：显示所有红包记录的总金额  
✅ **面额分布统计**：统计不同面额红包的数量（如：5元多少个）  
✅ **精美UI设计**：现代化响应式界面  
✅ **本地数据存储**：使用IndexedDB，数据安全可靠  
✅ **记录管理**：支持查看历史记录和删除单个记录  

## 🔧 技术说明
- 纯前端React应用，无需后端服务
- 所有数据存储在用户浏览器本地
- 支持响应式设计，适配各种设备
- 部署后立即可用
