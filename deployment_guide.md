# 服务器部署指南

## 项目信息
- **项目名称**：过年红包计数器
- **服务器IP**：175.178.84.235
- **构建目录**：`dist`
- **访问地址**：http://175.178.84.235

## 部署步骤

### 1. 上传文件到服务器
将以下文件上传到服务器Web根目录：
```
dist/
├── index.html          # 主页面
└── assets/
    ├── index-BU5Vx_w2.css  # 样式文件
    └── index-tBcCuiOO.js    # JavaScript文件
```

### 2. 配置Web服务器

#### 宝塔面板配置：
1. 登录宝塔面板
2. 进入网站管理
3. 选择现有网站或创建新网站
4. 设置网站根目录为 `dist` 文件夹位置
5. 确保访问路径正确配置

#### IIS配置：
1. 打开IIS管理器
2. 添加网站
3. 设置物理路径为 `dist` 文件夹
4. 绑定IP：175.178.84.235，端口80

### 3. 测试访问
- 浏览器访问：http://175.178.84.235
- 确保所有静态资源正确加载

## 功能特性
✅ 六个金额按钮：1元、5元、10元、20元、50元、100元  
✅ 实时汇总：显示所有红包记录的总金额  
✅ 面额统计：统计不同面额红包的数量分布  
✅ 精美UI：现代化响应式设计  
✅ 数据持久化：使用IndexedDB本地存储  
✅ 记录管理：支持查看历史记录和删除单个记录  

## 注意事项
1. 该项目为纯前端应用，无需后端服务
2. 所有数据存储在用户浏览器本地（IndexedDB）
3. 支持响应式设计，适配手机、平板、桌面设备
4. 确保服务器支持静态文件服务

## 文件说明
- `index.html` - 应用入口文件
- `assets/` - 构建后的静态资源文件
- 无需其他依赖，直接部署即可
